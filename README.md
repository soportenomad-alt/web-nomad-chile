# web-nomad-chile

## Carrito y compras (Snipcart)

1) En `index.html`, reemplaza `YOUR_SNIPCART_PUBLIC_API_KEY` por tu llave pública de Snipcart.
2) Configura moneda y pasarela en tu panel de Snipcart.
3) El catálogo se carga desde `data/catalogo.json` y las imágenes están en `assets/catalogo/images/`.
